Module xo_accel

Library for accesing the accelerometer of a XO 1.75.

The device will be named "xo_accel", module "xo_accel".

Functions

init (conf)
Initialize and starts the module. This is called automatically by toribio if the load attribute for the module in the configuration file is set to true.

Parameters:

  • conf the configuration table (see conf).
get_accel ()
Read the acceleration. The acceleration is measured in mg (1/1000th of earth gravity) the axis are,when looking from the front: x (horizontal to the right), y (horizontal to the front) and z (down)

Returns:

    The x, y and z magnitudes.
set_rate (hz)
Set the sensor sample rate. This is the internal sample rate, and the supported values are 1, 10, 25, 50, 100, 200, 400, 1600, and 5000 Hz. The driver has a limit on reading at about 25 Hz.

Parameters:

  • hz The rate in hz.
check ()
Run the sensor internal self test.

Returns:

    true on success or false,dx,dy,dz on failure.
